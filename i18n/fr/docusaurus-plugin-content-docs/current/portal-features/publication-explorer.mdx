# Explorateur de publications

![Explorateur de publications](./images/publication_explorer.png)

Vous pouvez explorer les publications et leurs détails qui ont été soumis au PSO. Veuillez noter que si une publication est sous embargo, seuls ses détails peuvent être consultés.

## Filtrage

Vous pouvez filtrer les publications par type à l’aide du **menu de filtrage des publications** situé du côté gauche de la page.

:::tip

Le tableau de filtrage est également disponible pour les rôles ayant accès à l’Explorateur régional des manuscrits !

:::

![Filtre](./images/publication_explorer_filter.png)

## Télécharger les informations sur les publications {/* #download-publications-information */}

Les informations sur les publications recherchées dans l’Explorateur de publications peuvent être téléchargées sous forme de fichier Microsoft Excel (`.xlsx`) ou de fichier [Research Information Systems (RIS)](https://fr.wikipedia.org/wiki/RIS_(format_de_fichier)) (`.ris`).

:::info
Si vous n’appliquez aucun filtre, les informations sur toutes les publications dans le PSO seront téléchargées !
:::
