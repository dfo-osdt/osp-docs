# 5.0 - Personnalisation et Sécurité

## 5.1 - Personnalisation

### 5.1.1 - Apparence

#### Langue

Pour basculer la langue affichée en français ou en anglais, cliquez sur le bouton ***FR/EN*** dans le coin supérieur droit de l'écran. Pour définir votre langue préférée, veuillez consulter [Langue par défaut](#_3-langue-par-defaut).

### 5.1.2 - Profil de l'utilisateur

#### Prénom et Nom

Le prénom et le nom associés à votre compte apparaîtront en gris. Si vous souhaitez modifier le nom associé à votre compte :
1. Cliquez sur le ***Champ du nom*** pour le sélectionner.
2. Appuyez sur la touche Retour arrière pour supprimer l'ancien nom.
3. Tapez le nouveau nom.
4. Cliquez sur le bouton ***Enregistrer*** pour sauvegarder les modifications.

#### Courriel

Le courriel associé à votre compte apparaîtra en gris. Ce courriel est également utilisé pour vous connecter à votre compte du Portail Science Ouverte. Si vous souhaitez modifier le courriel, veuillez envoyer un courriel à l'[Équipe de soutien du Portail Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

#### Langue par défaut

La langue par défaut pour votre compte apparaîtra dans cette boîte. Si vous souhaitez modifier votre langue par défaut :
1. Cliquez sur la ***Boîte de langue par défaut*** pour développer le menu déroulant.
2. Cliquez sur la langue désirée.
3. Cliquez sur le bouton ***Enregistrer*** pour sauvegarder les modifications.

#### Rôles des utilisateurs

Les rôles des utilisateurs identifient le type de compte que vous possédez. Les rôles des utilisateurs sont générés lors de la création du compte. Si vous pensez que votre compte n'a pas le rôle approprié, veuillez envoyer un courriel à l'[Équipe de soutien du Portail Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).
Rôles disponibles actuellement :
- Auteur
- Directeur
- Admin

### 5.1.3 - Profil de l'auteur

#### Affiliation actuelle

Si votre affiliation actuelle est incorrecte ou non renseignée, vous pouvez la mettre à jour en suivant les étapes suivantes :
1) Cliquez sur la boîte de texte Affiliation actuelle pour la sélectionner.
2) Supprimez l'affiliation existante en utilisant la touche Retour arrière.
3) Commencez à taper le nom de votre affiliation actuelle. Le système suggérera des organisations en fonction de votre saisie.
4) Si votre organisation apparaît, vous pouvez la sélectionner en cliquant sur son nom dans le menu déroulant.
5) Si votre organisation n'apparaît pas, vous pouvez l'ajouter en cliquant sur le bouton Plus ***+*** situé en bas à droite du menu déroulant.
6) La fenêtre contextuelle Créer une nouvelle organisation apparaîtra. Saisissez les informations suivantes de l'organisation :
    - Nom en anglais
    - Nom en français
    - Acronyme en anglais (optionnel)
    - Acronyme en français (optionnel)

#### ORCID

La boîte ORCID affichera votre identifiant ouvert des chercheurs et contributeurs si celui-ci est présent. Si aucun ORCID n'est associé à votre compte, cette boîte sera vide. Pour ajouter votre ORCID :
1. Cliquez sur la ***Boîte de texte ORCID*** pour la sélectionner.
2. Tapez votre ORCID.
3. Cliquez sur le bouton ***ENREGISTRER***.

#### Domaines d'expertise

La boîte Domaines d'expertise affichera les expertises que vous avez sélectionnées et qui représentent le mieux vos connaissances et votre expérience. Pour ajouter ou supprimer des expertises :
1. Cliquez sur le bouton ***MODIFIER***.
2. Cliquez sur la ***Boîte de texte des domaines d'expertise*** pour la sélectionner.
3. Commencez à taper l'expertise souhaitée.
4. Cliquez sur l'expertise que vous souhaitez ajouter dans la liste filtrée.
5. Pour supprimer une expertise, cliquez sur le bouton ***Icône X*** situé à droite du nom de l'expertise.
6. Cliquez sur le bouton ***ENREGISTRER*** pour sauvegarder les modifications.

#### Gérer ORCID

Pour plus d'informations sur la gestion de votre ORCID, veuillez consulter la section [6.1.3 - Gérer ORCID](/fr/guide/advanced-features.html#_6-1-3-gerer-orcid).

## 5.2 - Sécurité du compte

### 5.2.1 - Vérification du courriel

::: warning
Si le réseau est occupé, il peut falloir quelques minutes pour recevoir le courriel de vérification !
:::

Tous les comptes doivent être vérifiés lors de leur création. Un courriel contenant des instructions et un lien vers la page de vérification du Portail Science Ouverte est envoyé à l'adresse courriel associée au compte nouvellement créé.

Si vous rencontrez des problèmes pour vérifier votre compte, veuillez contacter l'[Équipe de soutien du Portail Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

### 5.2.2 - Mots de passe

#### Comment changer votre mot de passe

::: tip
Le Portail Science Ouverte a renforcé les exigences en matière de mots de passe. Les mots de passe doivent :
- contenir 12 caractères ou plus
- ne pas apparaître dans la liste des mots de passe compromis
:::

1. Dans le coin supérieur droit de votre écran, cliquez sur le bouton circulaire ***Menu utilisateur*** pour l'agrandir.
2. Cliquez sur le bouton ***Paramètres***.
3. Dans le ***Menu des paramètres*** sur le côté gauche de votre écran, cliquez sur le bouton ***Sécurité du compte***.
4. Dans la section ***Changer votre mot de passe***, cliquez sur la ***Boîte de texte du mot de passe actuel*** pour la sélectionner. Tapez votre mot de passe actuel.
5. Cliquez sur la ***Boîte de texte du nouveau mot de passe*** pour la sélectionner. Tapez le nouveau mot de passe souhaité.
6. Cliquez sur la ***Boîte de texte de confirmation du nouveau mot de passe*** pour la sélectionner. Tapez à nouveau votre nouveau mot de passe pour confirmer l'orthographe.
7. Cliquez sur le bouton ***CHANGER LE MOT DE PASSE***.

#### Réinitialiser le mot de passe

Si vous n'avez plus accès à votre mot de passe, vous pouvez demander une réinitialisation via le système automatique de réinitialisation de mot de passe :
1. Allez sur [osp-pso.mar.dfo-mpo.ca](https://osp-pso.mar.dfo-mpo.ca/#/).
2. Cliquez sur le bouton ***CONNEXION***.
3. Cliquez sur le bouton ***Mot de passe oublié ?***.
4. Tapez le courriel associé à votre compte et cliquez sur le bouton ***RÉINITIALISER***. Vous recevrez un courriel avec des instructions sur la façon de réinitialiser et de changer votre mot de passe.

Si vous n'avez pas reçu le courriel de réinitialisation de mot de passe ou si vous n'avez plus accès au courriel associé à votre compte, veuillez contacter l'[Équipe de soutien du Portail Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

### 5.2.3 - Historique d'authentification

Le Portail Science Ouverte conserve un historique des tentatives de connexion pour chaque compte utilisateur. L'Historique d'authentification visualise ces tentatives de connexion afin de vous permettre de vous assurer que des acteurs malveillants n'essaient pas d'accéder à votre compte.

::: danger
Si plusieurs tentatives de connexion se produisent en très peu de temps ou si d'autres activités suspectes sont observées, veuillez changer le mot de passe de votre compte Portail Science Ouverte et en informer l'[Équipe de soutien du Portail Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).
:::

#### Décomposition de l'historique d'authentification

![Tableau de l'historique d'authentification](/images/authentication_history_table.png)

##### 1 - Adresse IP

La colonne Adresse IP indique l'adresse de localisation internet associée à l'ordinateur tentant de se connecter à votre compte.

##### 2 - Agent

La colonne Agent indique le système d'exploitation de l'ordinateur tentant de se connecter à votre compte.

##### 3 - Localisation

::: conseil
Si vous vous connectez au Portail Science Ouverte via un VPN, cette localisation peut être différente de votre localisation actuelle.
:::

La colonne Localisation indique la localisation globale approximative associée à l'adresse IP.

##### 4 - Connexion à

La colonne Connexion à indique la date et l'heure à laquelle une tentative de connexion à votre compte a été effectuée. L'heure affichée sera basée sur l'heure de votre ordinateur.

##### 5 - Réussi

La colonne Réussi indique si votre compte a été connecté avec succès ou non. Si la connexion a réussi, une coche verte sera affichée. Si la connexion a échoué, une croix rouge sera affichée.

##### 6 - Déconnexion à

La colonne Déconnexion à indique la date et l'heure à laquelle votre compte a été déconnecté. L'heure affichée sera basée sur l'heure de votre ordinateur.
