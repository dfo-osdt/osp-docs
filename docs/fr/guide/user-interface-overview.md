# 3.0 - Aperçu de l'interaction utilisateur

## 3.1 - Page d'enregistrement

<!-- ![Page d'enregistrement](/images/register_help.png)-->
<img src="/images/register_help.png" width="500" height="500">

### 1 - Champ de texte Prénom

Cliquer sur le champ de texte Prénom vous permettra de le sélectionner. Une fois sélectionné, vous pouvez taper votre prénom.

### 2 - Champ de texte Nom

Cliquer sur le champ de texte Nom vous permettra de le sélectionner. Une fois sélectionné, vous pouvez taper votre nom de famille.

### 3 - Champ de texte Courriel

Cliquer sur le champ de texte Courriel vous permettra de le sélectionner. Une fois sélectionné, vous pouvez commencer à taper votre courriel professionnel.

### 4 - Champ de texte Mot de passe

Cliquer sur le champ de texte Mot de passe vous permettra de le sélectionner. Une fois sélectionné, vous pouvez taper votre mot de passe souhaité. Par défaut, les caractères de votre mot de passe seront masqués pendant que vous les tapez.\
Les mots de passe doivent répondre à certains critères pour garantir l'importance des identifiants de connexion sécurisés. Pour une liste complète de ces exigences, veuillez consulter [5.2.2 - Mots de passe](/fr/guide/customization-and-security.html#_5-2-securite-du-compte).

### 5 - Champ de texte Confirmation du mot de passe

Cliquer sur le champ de texte Confirmation du mot de passe vous permettra de le sélectionner. Une fois sélectionné, vous pouvez retaper votre mot de passe souhaité.

### 6 - Révéler le mot de passe

Cliquer sur le symbole de l'œil avec la ligne traversée vous permettra de visualiser temporairement ce qui a été tapé dans le champ de texte mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne traversée pendant la durée de l'affichage. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 7 - Bouton d'enregistrement

Cliquer sur le bouton d'enregistrement soumettra le formulaire d'inscription au système de création de compte. Si la création du compte est réussie, vous serez redirigé vers une page d'information avec les prochaines étapes pour vérifier votre compte.

## 3.2 - Page de connexion

<!-- ![Page de connexion](/images/login_help.png) -->
<img src="/images/login_help.png" width="500" height="500" alt="formulaire de connexion">

### 1 - Champ de texte Courriel

Cliquer sur le champ de texte Courriel vous permettra de le sélectionner. Une fois sélectionné, vous pouvez taper le courriel associé à votre compte.

Si vous ne vous souvenez pas du courriel associé à votre compte, veuillez consulter [7.1 - Problèmes courants](/fr/guide/troubleshooting.html#_7-1-problemes-courants).

### 2 - Champ de texte Mot de passe

Cliquer sur le champ de texte Mot de passe vous permettra de le sélectionner. Une fois sélectionné, vous pouvez commencer à taper le mot de passe associé à votre compte. Par défaut, les caractères de votre mot de passe seront masqués pendant que vous les tapez.\
Pour réinitialiser votre mot de passe, veuillez consulter la section [Mot de passe oublié](#_3-2-page-de-connexion).

### 3 - Révéler le mot de passe

Cliquer sur le symbole de l'œil avec la ligne traversée vous permettra de visualiser temporairement ce qui a été tapé dans le champ de texte mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne traversée pendant la durée de l'affichage. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 4 - Bouton de connexion

Cliquer sur le ***Bouton de connexion*** enverra les informations de courriel et mot de passe au service de connexion. Si la connexion réussit, vous serez automatiquement redirigé vers la page [Tableau de bord OSP](#_3-3-tableau-de-bord).

### 5 - Mot de passe oublié et pas de compte

#### Mot de passe oublié

Cliquer sur le bouton Mot de passe oublié vous mènera à la page de réinitialisation du mot de passe. À partir de là, vous pouvez cliquer sur ***Votre champ de texte courriel*** pour le sélectionner. Une fois sélectionné, vous pouvez taper le courriel associé à votre compte. Des instructions avec un lien hypertexte pour réinitialiser votre mot de passe seront envoyées à ce courriel.

::: warning
Si votre courriel n'a pas été utilisé pour enregistrer un compte, vous ne recevrez aucune communication de l'OSP !
:::

Pour plus d'informations sur la réinitialisation de votre mot de passe, veuillez consulter [5.2.2 - Mots de passe](/fr/guide/customization-and-security.html#_5-2-securite-du-compte). 

#### Pas de compte

Cliquer sur le ***Bouton Pas de compte ?*** vous amènera à la [page d'inscription](#_3-1-page-d-enregistrement).

#### Bouton Accueil en triangle

Cliquer sur le Triangle vous ramènera à la page d'accueil de l'OSP.

#### Bouton Circulaire avec initiales

### 3 - Menu contextuel

#### Bouton Tableau de bord

Cliquer sur le ***Bouton Tableau de bord*** vous ramènera à la page [Tableau de bord](#_3-2-page-de-connexion).

#### Bouton Mon Manuscrit

Cliquer sur le ***Bouton Mon Manuscrit*** vous amènera à la [page Mon Manuscrit](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs). À partir de là, vous pouvez créer, continuer ou consulter vos formulaires d'enregistrement de manuscrit (MRF).

Cliquer sur le ***Bouton Mes Publications*** vous amènera à la [page Mes Publications](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs). À partir de là, vous pouvez créer et modifier vos demandes de publication, consulter l'état des demandes de publication ou voir la liste de vos manuscrits publiés.

#### Bouton Mes Avis

Cliquer sur le ***Bouton Mes Avis*** vous amènera à la [page Mes Avis](/fr/guide/using-the-application.html#_4-3-guide-pour-les-gestionnaires). À partir de là, vous pouvez voir les avis nécessitant votre action, consulter l'état de vos avis en cours, ou voir la liste de vos avis complétés.

### 4 - Explorer

#### Bouton Publications

Cliquer sur le ***Bouton Publications*** vous amènera à la [page Mes Publications](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs). À partir de là, vous pouvez parcourir toutes les publications soumises via l'OSP.

#### Bouton Auteurs

Cliquer sur le ***Bouton Auteurs*** vous amènera à la page [Auteurs](#).

::: warning
La page Auteurs est actuellement en cours de construction.
:::

### 5 - Métriques de statut

#### Métriques Mon Manuscrit

Le cadre de métriques Mon Manuscrit montre le nombre de formulaires d'enregistrement de manuscrit en cours. En cliquant sur la flèche orientée vers la droite, le ***Bouton MRF*** situé dans le coin supérieur droit du cadre Mon Manuscrit, vous serez redirigé vers la [page Mon Manuscrit](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs).

#### Métriques Mes Avis

Le cadre de métriques Mes Avis montre le nombre de formulaires d'enregistrement de manuscrit en attente de révision par un gestionnaire. En cliquant sur la flèche orientée vers la droite, le ***Bouton Avis*** situé dans le coin supérieur droit du cadre Mes Avis, vous serez redirigé vers la [page Mes Avis](/fr/guide/using-the-application.html#_4-3-guide-pour-les-gestionnaires).

#### Métriques Mes Publications

Le cadre de métriques Mes Publications montre le nombre de manuscrits acceptés par un éditeur mais non encore publiés. En cliquant sur la flèche orientée vers la droite, le ***Bouton Publications*** situé dans le coin supérieur droit du cadre Mes Publications, vous serez redirigé vers la [page Mes Publications](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs).

### 6 - Récent

#### Manuscrits

![Tableau de bord Manuscrit récent](/images/dashboard_recent_manuscript_help.png)

L'onglet Manuscrits du menu Récent montre les formulaires d'enregistrement de manuscrit qui ont été modifiés le plus récemment. Ces formulaires sont classés du plus récemment modifié au moins récemment.

#### Avis

![Tableau de bord Avis récents](/images/dashboard_recent_reviews_help.png)

L'onglet Avis du menu Récent montre les formulaires d'enregistrement de manuscrit dont le statut a le plus récemment changé. Ces formulaires sont classés du plus récent changement de statut au plus ancien. Si des formulaires d'enregistrement de manuscrit attendent votre révision, une petite bulle avec le nombre d'avis en attente apparaîtra dans le coin supérieur droit de l'onglet Avis.

#### Publications

![Tableau de bord Publications récentes](/images/dashboard_recent_publications_help.png)

L'onglet Publications du menu Récent montre les publications dont le statut a le plus récemment changé et auxquelles vous êtes associé. Ces publications sont classées du plus récent changement de statut au plus ancien.

### 7 - Création rapide

#### Créer un manuscrit

En cliquant sur le ***Bouton Créer un Manuscrit***, une nouvelle [application de formulaire d'enregistrement de manuscrit](/fr/guide/using-the-application.html#_4-2-guides-pour-les-auteurs) s'ouvrira.

#### Ajouter une publication

En cliquant sur le ***Bouton Ajouter une Publication***, vous pourrez ajouter des publications qui ont été publiées avant ou sans l'utilisation du Portail de Science Ouverte. Ce bouton ouvrira le [formulaire de création d'une nouvelle entrée](/fr/guide/using-the-application.html#_4-2-2-page-de-mes-publications).

### 8 - Nous contacter

En cliquant sur le ***Lien Nous Contacter***, vous serez automatiquement invité à envoyer un email à `DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca` via votre client de messagerie par défaut.

### 3.4.2 - Profil Utilisateur

![Paramètres Profil Utilisateur](/images/settings_user_profile_help.png)

#### 1 - Prénom et Nom de Famille

Le prénom et le nom de famille associés à votre compte apparaîtront en gris. Si vous souhaitez modifier le nom associé à votre compte :
1. Cliquez sur le ***Champ Nom*** pour le sélectionner.
2. Appuyez sur la ***Touche Retour Arrière*** pour supprimer l'ancien nom.
3. Tapez le nouveau nom.
4. Cliquez sur le ***[Bouton Enregistrer](#_4-bouton-enregistrer)*** pour sauvegarder les modifications.

#### 2 - Courriel

Le courriel associé à votre compte apparaîtra en gris. Ce courriel est également utilisé pour vous connecter à votre compte OSP. Si vous souhaitez modifier le courriel, veuillez envoyer un courriel à l'[Équipe de Support du Portail de Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

#### 3 - Langue par Défaut

La langue par défaut de votre compte apparaîtra dans cette boîte. Si vous souhaitez changer la langue par défaut :
1. Cliquez sur la ***Boîte Langue par Défaut*** pour développer le menu déroulant.
2. Cliquez sur la langue souhaitée.
3. Cliquez sur le ***[Bouton Enregistrer](#_4-bouton-enregistrer)*** pour sauvegarder les modifications.

#### 4 - Bouton Enregistrer

Le ***Bouton Enregistrer*** est utilisé pour sauvegarder les modifications apportées au Profil Utilisateur. Pour appliquer les modifications, cliquez sur le ***Bouton Enregistrer***.

#### 5 - Rôles Utilisateurs

Les rôles utilisateurs identifient le type de compte et d'accès à l'application dont vous disposez. Le rôle **Auteur** est accordé à tous les comptes lors de leur création. Si vous estimez que votre compte n'a pas le rôle approprié, veuillez envoyer un courriel à l'[Équipe de Support du Portail de Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).\
Rôles disponibles :
- Auteur
- Directeur
- Admin

### 3.4.3 - Profil Auteur

![Paramètres Profil Auteur](/images/settings_author_profile_help.png)

#### 1 - Informations Utilisateur

Les informations utilisateur sont les mêmes que celles trouvées dans la section [3.4.2 - Profil Utilisateur](#_2-parametres). Si vous souhaitez modifier ces informations, veuillez naviguer vers les Paramètres du Profil Utilisateur.

#### 2 - Affiliation Actuelle

Le champ Affiliation Actuelle indique l'organisation à laquelle vous êtes actuellement affilié. Si vous souhaitez modifier votre affiliation actuelle :
1. Cliquez sur le ***Champ Affiliation Actuelle*** pour le sélectionner.
2. Appuyez sur la ***Touche Retour Arrière*** pour supprimer l'ancienne affiliation.
3. Tapez le nom de votre nouvelle affiliation. Le système tentera de filtrer les organisations déjà vues. Si votre affiliation n'est pas trouvée, vous pouvez l'ajouter :
    1. Cliquez sur le ***Bouton "+" Vert***.
    2. Remplissez les cases avec le nom de l'organisation en anglais puis en français.
    3. Cliquez sur le ***Bouton CRÉER***.
4. Cliquez sur le ***Nom de l'Organisation*** dans la liste filtrée.
5. Cliquez sur le ***Bouton ENREGISTRER***.

#### 3 - ORCID

La boîte ORCID affichera votre Open Research and Contributor ID s'il est présent. Si aucun ORCID n'est associé à votre compte, cette boîte sera vide. Pour ajouter votre ORCID :
1. Cliquez sur la ***Boîte de Texte*** pour la sélectionner.
2. Saisissez votre ORCID.
3. Cliquez sur le ***Bouton ENREGISTRER***.

#### 4 - Bouton Enregistrer

Le ***Bouton Enregistrer*** confirmera toutes les modifications apportées à la section Profil Auteur. Le ***Bouton Enregistrer*** est activé en cliquant dessus.

#### 5 - Domaines d'Expertise

La boîte Domaines d'Expertise affichera les expertises que vous avez sélectionnées et qui, selon vous, représentent le mieux vos connaissances et votre expérience. Pour savoir comment ajouter ou supprimer des Domaines d'Expertise, veuillez consulter [6 - Bouton Modifier](#_6-bouton-modifier).

#### 6 - Bouton Modifier

Le ***Bouton Modifier*** vous permet d'ajouter ou de supprimer des [Domaines d'Expertise](#_5-domaines-d-expertise). Pour ajouter ou supprimer une expertise :
1. Cliquez sur le ***Bouton MODIFIER***.
2. Cliquez sur la ***Boîte de Texte*** pour la sélectionner.
3. Commencez à taper l'expertise souhaitée.
4. Cliquez sur le ***Nom de l'Expertise*** que vous souhaitez ajouter dans la liste filtrée.
5. Pour supprimer une expertise, cliquez sur le ***Bouton "X"*** situé à droite du nom de l'expertise.
6. Cliquez sur le ***Bouton ENREGISTRER***.

#### 7 - Gérer ORCID

La section Gérer ORCID vous fournit un bouton pour enregistrer ou vérifier votre ORCID. Pour en savoir plus sur ORCID, veuillez consulter [6.1 - Open Researcher and Contributor Identifier](/fr/guide/advanced-features.html#_6-1-identifiant-ouvert-des-chercheurs-et-contributeurs).

### 3.4.4 - Sécurité du Compte

![Paramètres Sécurité du Compte](/images/settings_account_security_help.png)

#### 1 - Mot de Passe Actuel

La boîte Mot de Passe Actuel est une boîte de texte pour entrer votre mot de passe actuel si vous essayez de le changer.

#### 2 - Nouveau Mot de Passe

La boîte Nouveau Mot de Passe est une boîte de texte pour entrer votre nouveau mot de passe souhaité. Ce mot de passe doit également être saisi dans la boîte Confirmer le Nouveau Mot de Passe pour garantir l'exactitude. Pour plus d'informations sur la modification de votre mot de passe, veuillez consulter [Section 5.2.2 - Mots de Passe](/fr/guide/customization-and-security.html#_5-2-securite-du-compte).

#### 3 - Bouton Afficher le Mot de Passe

Le Bouton Afficher le Mot de Passe affichera brièvement le texte présent dans la boîte de texte Mot de Passe associée. Les ***Boutons Afficher le Mot de Passe*** sont activés en cliquant dessus.

#### 4 - Bouton Changer le Mot de Passe

Le ***Bouton Changer le Mot de Passe*** est le bouton utilisé pour soumettre une demande de changement de mot de passe. Ce bouton est activé en cliquant dessus.

#### 5 - Historique d'Authentification

Le tableau Historique d'Authentification montre toutes les tentatives de connexion à votre compte. Pour plus d'informations sur l'Historique d'Authentification, veuillez consulter [Section 5.2.3 - Historique d'Authentification](/fr/guide/customization-and-security.html#_5-2-3-historique-d-authentification).

##### Adresse IP

La colonne Adresse IP représente l'adresse de localisation internet associée à l'ordinateur essayant de se connecter à votre compte.

##### Agent

La colonne Agent représente le système d'exploitation de l'ordinateur essayant de se connecter à votre compte.

##### Localisation

La colonne Localisation est l'emplacement global approximatif associé à l'Adresse IP.

##### Connexion À

La colonne Connexion À montre la date et l'heure à laquelle une tentative de connexion à votre compte a été effectuée. L'heure affichée sera basée sur l'heure de votre ordinateur.

##### Réussite

La colonne Réussite indique si votre compte a été connecté avec succès ou non. Si la connexion est réussie, une coche verte sera affichée. Si elle échoue, une croix rouge sera affichée.

##### Déconnexion À

La colonne Déconnexion À montre la date et l'heure à laquelle votre compte a été déconnecté. L'heure affichée sera basée sur l'heure de votre ordinateur.

### 3.4.5 - Invitations Envoyées

![Paramètres Invitations Envoyées](/images/settings_sent_invitations_help.png)

#### 1 - Invitations Envoyées

Les invitations à la PSO (Portail Science Ouverte) que vous avez envoyées à des collègues seront affichées dans le tableau des Invitations Envoyées.

##### Statut

La colonne Statut affichera l'état actuel de l'invitation.

##### Utilisateur

La colonne Utilisateur affichera le nom de l'utilisateur que vous avez invité.

##### Courriel

La colonne Courriel affichera l'adresse électronique à laquelle l'invitation a été envoyée.

##### Invitation Envoyée Le

La colonne Invitation Envoyée Le affichera la date et l'heure auxquelles l'invitation a été envoyée. La date et l'heure seront basées sur l'horloge de votre ordinateur.

##### Inscrit Le

La colonne Inscrit Le affichera la date et l'heure auxquelles la personne que vous avez invitée a créé son compte sur le Portail Science Ouverte. La date et l'heure seront basées sur l'horloge de votre ordinateur.
