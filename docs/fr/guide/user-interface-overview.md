# 3.0 - Présentation de l'Interface Utilisateur

## 3.1 - Page d'Inscription
<!-- ![Page d'Inscription](/images/register_help.png)-->
<img src="/images/register_help.png" width="500" height="500">

### 1 - Champ de Prénom
En cliquant sur le champ de prénom, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez saisir votre prénom.

### 2 - Champ de Nom de Famille
En cliquant sur le champ de nom de famille, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez saisir votre nom de famille.

### 3 - Champ d'Adresse E-mail
En cliquant sur le champ d'adresse e-mail, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez commencer à saisir l'e-mail que vous souhaitez associer à votre compte. Il est fortement recommandé de créer votre compte avec votre adresse e-mail DFO-MPO.

### 4 - Champ de Mot de Passe
En cliquant sur le champ de mot de passe, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez saisir le mot de passe souhaité. Par défaut, les caractères de votre mot de passe seront masqués pendant que vous les saisissez.\
Les mots de passe doivent répondre à certains critères pour promouvoir l'importance des informations d'identification de connexion sécurisées. Pour obtenir la liste complète de ces exigences, veuillez visiter [Personnalisation et Sécurité: 5.2 - Mots de Passe](/guide/customization-and-security.html#_5-2-passwords).

### 5 - Confirmer le Mot de Passe
En cliquant sur le champ de confirmation de mot de passe, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez ressaisir le mot de passe désiré.

### 6 - Révéler le Mot de Passe
En cliquant sur le symbole de l'œil avec la ligne barrée, vous pouvez temporairement voir ce qui a été saisi dans le champ de texte du mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne barrée pendant la durée de la révélation. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 7 - Bouton d'Inscription
En cliquant sur le bouton d'inscription, vous soumettez le formulaire d'inscription au système de création de compte. Si la création du compte réussit, vous serez redirigé vers une page d'information avec les prochaines étapes pour vérifier votre compte.

## 3.2 - Page de Connexion
<!-- ![Page de Connexion](/images/login_help.png) -->
<img src="/images/login_help.png" width="500" height="500">

### 1 - Champ d'Adresse E-mail
En cliquant sur le champ d'adresse e-mail, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez saisir l'e-mail associé à votre compte.\
Si vous ne vous souvenez pas de l'e-mail associé à votre compte, veuillez consulter la page [Dépannage: 7.2 - Connexion](/guide/troubleshooting.html#_7-2-logging-in).

### 2 - Champ de Mot de Passe
En cliquant sur le champ de mot de passe, vous pouvez le sélectionner. Une fois sélectionné, vous pouvez commencer à saisir le mot de passe associé à votre compte. Par défaut, les caractères de votre mot de passe seront masqués pendant que vous les saisissez.\
Pour réinitialiser votre mot de passe, veuillez consulter la section [Mot de Passe Oublié](#forgot-password).

### 3 - Révéler le Mot de Passe
En cliquant sur le symbole de l'œil avec la ligne barrée, vous pouvez temporairement voir ce qui a été saisi dans le champ de texte du mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne barrée pendant la durée de la révélation. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 4 - Bouton de Connexion
En cliquant sur le bouton "CONNEXION", les identifiants d'e-mail et de mot de passe seront envoyés au service de connexion. En cas de succès, vous serez automatiquement redirigé vers la page du [Tableau de Bord du Portail de la Science Ouverte](#_3-2-dashboard).\
Si la vérification de connexion échoue, une bannière rouge apparaîtra sous le titre "Connexion" avec une brève description de la raison de l'échec de la vérification. Si ce problème persiste, veuillez consulter la page [Dépannage: 7.2 - Connexion](/guide/troubleshooting.html#_7-2-3-error-messages).

### 5 - Boutons de Mot de Passe Oublié et Création de Compte
#### Mot de Passe Oublié
En cliquant sur le bouton Mot de Passe Oublié, vous serez dirigé vers la page de Réinitialisation de Mot de Passe. De là, vous pouvez cliquer sur le champ "Votre Email" pour le sélectionner. Une fois sélectionné, vous pouvez saisir l'e-mail associé à votre compte. Des instructions avec un lien hypertexte pour réinitialiser votre mot de passe seront envoyées à cet e-mail.\
Pour plus d'informations sur la réinitialisation de votre mot de passe ou de votre e-mail, veuillez visiter la page [Dépannage: 7.2 - Connexion](/guide/troubleshooting.html#_7-2-logging-in).

La section Publications du menu Récent affiche les publications dont le statut a récemment changé et auxquelles vous êtes associé. Ces publications sont classées de manière décroissante en fonction de la date du changement de statut.

### 3.3.2 - Profil de l'Utilisateur
![Profil de l'Utilisateur](/images/settings_user_profile_help.png)

#### 1 - Prénom et Nom de Famille
Le prénom et le nom de famille associés à votre compte apparaîtront en gris. Si vous souhaitez changer le nom associé à votre compte :
1. Cliquez sur le champ du nom pour le sélectionner.
2. Appuyez sur la touche de retour arrière pour supprimer l'ancien nom.
3. Tapez le nouveau nom.
4. Cliquez sur le [Bouton Enregistrer](#_4-save-button) pour sauvegarder les modifications.

#### 2 - Email
L'email associé à votre compte apparaîtra en gris. Cet email est également utilisé pour vous connecter à votre compte du Portail de la Science Ouverte. Si vous souhaitez apporter des modifications à votre email, veuillez contacter l'équipe de support du Portail de la Science Ouverte à l'adresse suivante : [DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

#### 3 - Langue par Défaut
La langue par défaut de votre compte apparaîtra dans cette boîte. Si vous souhaitez changer votre langue par défaut :
1. Cliquez sur la boîte de langue par défaut pour déployer le menu déroulant.
2. Sélectionnez la langue désirée dans le menu déroulant.
3. Cliquez sur le [Bouton Enregistrer](#_4-save-button) pour sauvegarder les modifications.

#### 4 - Bouton Enregistrer
Le Bouton Enregistrer est utilisé pour sauvegarder toutes les modifications apportées au Profil de l'Utilisateur. Pour appliquer les modifications, cliquez sur le Bouton Enregistrer.

#### 5 - Rôles de l'Utilisateur
Les Rôles de l'Utilisateur identifient le type de compte que vous avez. Les Rôles de l'Utilisateur sont générés lors de la création du compte. Si vous estimez que votre compte n'a pas le rôle approprié, veuillez contacter l'équipe de support du Portail de la Science Ouverte à l'adresse suivante : [DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).\
Rôles disponibles actuellement :
- Auteur
- Directeur
- Administrateur

### 3.3.3 - Profil de l'Auteur
![Profil de l'Auteur](/images/settings_author_profile_help.png)

#### 1 - Informations Utilisateur
Les Informations Utilisateur sont les mêmes que celles trouvées dans la section [3.3.2 - Profil Utilisateur](#3-3-2-profil-de-lutilisateur). Si vous souhaitez apporter des modifications à ces informations, veuillez naviguer vers les Paramètres de Profil Utilisateur.

#### 2 - Affiliation Actuelle
La boîte d'Affiliation Actuelle indique l'organisation à laquelle vous êtes actuellement affilié. Si vous souhaitez changer votre affiliation actuelle :
1. Cliquez sur le champ d'Affiliation Actuelle pour le sélectionner.
2. Appuyez sur la touche de retour arrière pour supprimer l'ancienne affiliation.
3. Tapez le nom de votre nouvelle affiliation. Le système tentera de filtrer les organisations précédemment consultées. Si votre affiliation n'est pas trouvée, vous pouvez l'ajouter :
    1. Cliquez sur l'icône verte "+".
    2. Remplissez les cases avec le nom anglais puis français de l'organisation.
    3. Cliquez sur le bouton "CREATE".
4. Cliquez sur le nom de l'organisation dans la liste filtrée.
5. Cliquez sur le bouton "SAVE".

#### 3 - ORCID
La boîte ORCID affichera votre identifiant de chercheur et contributeur ouvert si présent. Si aucun ORCID n'est associé à votre compte, cette boîte sera vide. Pour ajouter votre ORCID :
1. Cliquez sur la boîte de texte pour la sélectionner.
2. Tapez votre ORCID.
3. Cliquez sur le bouton "SAVE".

#### 4 - Bouton Enregistrer
Le bouton Enregistrer confirmera les modifications apportées à votre section Profil de l'Auteur. Le bouton Enregistrer est activé en cliquant dessus.

#### 5 - Domaines d'Expertise
La boîte Domaines d'Expertise affichera les domaines dans lesquels vous avez indiqué avoir des compétences et des connaissances. Pour plus d'informations sur la façon d'ajouter ou de supprimer des domaines d'expertise, veuillez consulter [6 - Bouton Éditer](#_6-bouton-editer).

#### 6 - Bouton Éditer
Le bouton Éditer vous permet d'ajouter ou de supprimer des [Domaines d'Expertise](#_5-domaines-dexpertise). Pour ajouter ou supprimer une expertise :
1. Cliquez sur le bouton "ÉDITER".
2. Cliquez sur la boîte de texte pour la sélectionner.
3. Commencez à taper le domaine d'expertise souhaité.
4. Cliquez sur le domaine d'expertise que vous souhaitez ajouter dans la liste filtrée.
5. Pour supprimer une expertise, cliquez sur le bouton "X" situé à droite du nom de l'expertise.
6. Cliquez sur le bouton "SAVE" pour enregistrer les modifications.

#### 7 - Gérer l'ORCID
La section Gérer l'ORCID vous propose un bouton pour enregistrer ou vérifier votre ORCID. Pour en savoir plus sur l'ORCID, veuillez consulter [ORCID dans la Section 10 - Appendices](/guide/appendices.html).

### 3.3.4 - Sécurité du Compte
![Paramètres Sécurité du Compte](/images/settings_account_security_help.png)

#### 1 - Mot de Passe Actuel
La boîte de Mot de Passe Actuel est un champ de texte pour entrer votre mot de passe actuel si vous essayez de le changer.

#### 2 - Nouveau Mot de Passe
La boîte de Nouveau Mot de Passe est un champ de texte pour entrer votre nouveau mot de passe désiré. Ce mot de passe doit également être entré dans la boîte de Confirmation du Nouveau Mot de Passe pour assurer l'orthographe correcte. Pour plus d'informations sur la modification de votre mot de passe, veuillez consulter [Section 5.2 - Mots de Passe](/guide/customization-and-security.html#_5-2-passwords).

#### 3 - Bouton Afficher le Mot de Passe
Le Bouton Afficher le Mot de Passe affichera brièvement le texte qui se trouve dans la boîte de texte de Mot de Passe associée au bouton d'affichage. Les boutons Afficher le Mot de Passe sont activés en cliquant dessus.

#### 4 - Bouton Modifier le Mot de Passe
Le Bouton Modifier le Mot de Passe est le bouton utilisé pour soumettre une demande de changement de mot de passe. Ce bouton est activé en cliquant dessus.

#### 5 - Historique d'Authentification
Le tableau Historique d'Authentification affiche toutes les tentatives de connexion à votre compte. Pour plus d'informations sur l'Historique d'Authentification, veuillez consulter [Section 5.2.3 - Historique d'Authentification](/guide/customization-and-security.html#_5-2-3-authentication-history).

##### Adresse IP
La colonne Adresse IP est l'adresse de localisation Internet associée à l'ordinateur qui tente de se connecter à votre compte.

##### Agent
La colonne Agent est le système d'exploitation de l'ordinateur qui tente de se connecter à votre compte.

##### Lieu
La colonne Lieu est le lieu global approximatif associé à l'adresse IP.

##### Connexion à
La colonne Connexion à est la date et l'heure à laquelle une tentative de connexion à votre compte a été effectuée. L'heure indiquée sera basée sur l'heure de votre ordinateur.

##### Réussie
La colonne Réussie indique si votre compte a été connecté avec succès ou non. En cas de succès, une coche verte sera affichée. En cas d'échec, une croix rouge sera affichée.

##### Déconnexion à
La colonne Déconnexion à indique la date et l'heure à laquelle votre compte a été déconnecté. L'heure indiquée sera basée sur l'heure de votre ordinateur.

Le bouton Enregistrer enverra les modifications que vous avez apportées au serveur de l'Open Science Portal. Le symbole sur le bouton d'enregistrement est une disquette.

### 3.4.13 - Bouton de Révélation de Texte
Le bouton de Révélation de Texte affichera brièvement votre texte de mot de passe afin que vous puissiez vérifier l'orthographe.
