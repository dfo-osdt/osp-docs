# 3.0 - Aperçu de l'Interface Utilisateur

## 3.1 - Page d'Inscription
<!-- ![Page d'Inscription](/images/register_help.png)-->
<img src="/images/register_help.png" width="500" height="500">

### 1 - Champ Prénom
En cliquant sur le champ Prénom, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez saisir votre prénom.

### 2 - Champ Nom de Famille
En cliquant sur le champ Nom de Famille, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez saisir votre nom de famille.

### 3 - Champ Email
En cliquant sur le champ Email, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez commencer à saisir l'email que vous souhaitez associer à votre compte. Il est fortement recommandé de créer votre compte avec votre adresse email DFO-MPO.

### 4 - Champ Mot de Passe
En cliquant sur le champ Mot de Passe, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez saisir votre mot de passe souhaité. Par défaut, les caractères de votre mot de passe seront masqués lorsque vous les saisissez.\
Les mots de passe doivent répondre à certains critères pour promouvoir l'importance de la sécurisation des informations de connexion. Pour une liste complète de ces exigences, veuillez visiter [5.2.2 - Mots de Passe](/fr/guide/customization-and-security.html#_5-2-2-mots-de-passe).

### 5 - Confirmation du Mot de Passe
En cliquant sur le champ Confirmation du Mot de Passe, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez ressaisir votre mot de passe souhaité.

### 6 - Afficher le Mot de Passe
En cliquant sur le symbole de l'œil avec une ligne barrée à travers, vous pourrez temporairement voir ce qui a été saisi dans le champ de texte du mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne barrée pendant la durée de la révélation. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 7 - Bouton Inscription
En cliquant sur le bouton Inscription, vous soumettrez le formulaire d'inscription au système de création de compte. Si la création de compte est réussie, vous serez redirigé vers une page d'information avec les prochaines étapes pour vérifier votre compte.

## 3.2 - Page de Connexion
<!-- ![Page de Connexion](/images/login_help.png) -->
<img src="/images/login_help.png" width="500" height="500">

### 1 - Champ Email
En cliquant sur le champ Email, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez saisir l'email associé à votre compte.

Si vous ne vous souvenez pas de l'email associé à votre compte, veuillez consulter [7.1 - Problèmes Courants](/fr/guide/troubleshooting.html#je-ne-connais-pas-l-email-associe-a-mon-compte).

### 2 - Champ Mot de Passe
En cliquant sur le champ Mot de Passe, vous pourrez le sélectionner. Une fois sélectionné, vous pouvez commencer à saisir le mot de passe associé à votre compte. Par défaut, les caractères de votre mot de passe seront masqués lorsque vous les saisissez.\
Pour réinitialiser votre mot de passe, veuillez consulter la section [Mot de Passe Oublié](#mot-de-passe-oublie).

### 3 - Afficher le Mot de Passe
En cliquant sur le symbole de l'œil avec une ligne barrée à travers, vous pourrez temporairement voir ce qui a été saisi dans le champ de texte du mot de passe pendant cinq (5) secondes. Le symbole de l'œil perdra sa ligne barrée pendant la durée de la révélation. Si vous souhaitez masquer votre mot de passe après l'avoir révélé, cliquez à nouveau sur le symbole de l'œil.

### 4 - Bouton Connexion
En cliquant sur le bouton "LOGIN", les informations d'identification email et mot de passe saisies seront envoyées au service de connexion. Si la connexion réussit, vous serez automatiquement redirigé vers la page du [Tableau de Bord du Portail de la Science Ouverte](#_3-2-tableau-de-bord).

### 5 - Boutons Mot de Passe Oublié et Création de Compte
#### Mot de Passe Oublié
En cliquant sur le bouton Mot de Passe Oublié, vous serez redirigé vers la page de Réinitialisation de Mot de Passe. À partir de là, vous pouvez cliquer sur le champ de texte "Votre Email" pour le sélectionner. Une fois sélectionné, vous pouvez saisir l'email associé à votre compte. Des instructions avec un lien hypertexte pour réinitialiser votre mot de passe seront envoyées à cet email.

Pour plus d'informations sur la réinitialisation de votre mot de passe, veuillez visiter [5.2.2 - Mots de Passe](/fr/guide/customization-and-security.html#reinitialiser-le-mot-de-passe).

### 3.3.2 - Profil Utilisateur
![Paramètres Profil Utilisateur](/images/settings_user_profile_help.png)

#### 1 - Prénom et Nom de Famille
Le prénom et le nom de famille associés à votre compte apparaîtront en gris. Si vous souhaitez modifier le nom associé à votre compte :
1. Cliquez sur le champ du nom pour le sélectionner.
2. Appuyez sur la touche de retour arrière pour supprimer l'ancien nom.
3. Tapez le nouveau nom.
4. Cliquez sur le [bouton Enregistrer](#_4-bouton-enregistrer) pour sauvegarder les modifications.

#### 2 - Email
L'email associé à votre compte apparaîtra en gris. Cet email est également utilisé pour vous connecter à votre compte Open Science Portal. Si vous souhaitez apporter des modifications à l'email, veuillez envoyer un email à l'[équipe de support du Portail de la Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).

#### 3 - Langue par Défaut
La langue par défaut de votre compte apparaîtra dans cette case. Si vous souhaitez changer votre langue par défaut :
1. Cliquez sur la case Langue par Défaut pour dérouler le menu déroulant.
2. Cliquez sur la langue de votre choix.
3. Cliquez sur le [bouton Enregistrer](#_4-bouton-enregistrer) pour sauvegarder les modifications.

#### 4 - Bouton Enregistrer
Le bouton Enregistrer est utilisé pour sauvegarder les modifications apportées au Profil Utilisateur. Pour appliquer toutes les modifications, cliquez sur le bouton Enregistrer.

#### 5 - Rôles Utilisateur
Les rôles utilisateur identifient le type de compte que vous possédez. Les rôles utilisateur sont générés lors de la création du compte. Si vous pensez que votre compte n'a pas le rôle approprié, veuillez envoyer un email à l'[équipe de support du Portail de la Science Ouverte](mailto:DFO.OpenScience-ScienceOuverte.MPO@dfo-mpo.gc.ca).
Rôles disponibles actuellement :
- Auteur
- Directeur
- Administrateur

### 3.3.3 - Profil Auteur
![Paramètres Profil Auteur](/images/settings_author_profile_help.png)

#### 1 - Informations Utilisateur
Les Informations Utilisateur sont les mêmes que celles trouvées dans la section [3.3.2 - Profil Utilisateur](#_3-3-2-profil-utilisateur). Si vous souhaitez apporter des modifications à ces informations, veuillez vous rendre dans les Paramètres du Profil Utilisateur.

#### 2 - Affiliation Actuelle
La boîte d'Affiliation Actuelle indique l'organisation à laquelle vous êtes actuellement affilié. Si vous souhaitez changer votre affiliation actuelle :
1. Cliquez sur le champ d'Affiliation Actuelle pour le sélectionner.
2. Appuyez sur la touche de retour arrière pour supprimer l'ancienne affiliation.
3. Tapez le nom de votre nouvelle affiliation. Le système tentera de filtrer les organisations précédemment vues. Si votre affiliation n'est pas trouvée, vous pouvez l'ajouter :
    1. Cliquez sur l'icône verte "+".
    2. Remplissez les cases avec le nom anglais puis français de l'organisation.
    3. Cliquez sur le bouton "CRÉER".
4. Cliquez sur le nom de l'organisation dans la liste filtrée.
5. Cliquez sur le bouton "ENREGISTRER".

#### 3 - ORCID
La boîte ORCID affichera votre Identifiant de Chercheur et Contributeur Ouvert (ORCID) s'il est présent. Si aucun ORCID n'est associé à votre compte, cette boîte sera vide. Pour ajouter votre ORCID :
1. Cliquez sur la boîte de texte pour la sélectionner.
2. Tapez votre ORCID.
3. Cliquez sur le bouton "ENREGISTRER".

#### 4 - Bouton Enregistrer
Le bouton Enregistrer confirmera les modifications apportées à votre Profil Auteur. Le bouton Enregistrer est activé en le cliquant.

#### 5 - Domaines d'Expertise
La boîte Domaines d'Expertise affichera les domaines dans lesquels vous avez sélectionné que vous vous sentez le mieux qualifié. Pour des informations sur la façon d'ajouter ou de supprimer des domaines d'expertise, veuillez consulter [6 - Bouton Éditer](#_6-bouton-editer).

#### 6 - Bouton Éditer
Le Bouton Éditer vous permet d'ajouter ou de supprimer des [Domaines d'Expertise](#_5-domaines-d-expertise). Pour ajouter ou supprimer une expertise :
1. Cliquez sur le bouton "ÉDITER".
2. Cliquez sur la boîte de texte pour la sélectionner.
3. Commencez à taper l'expertise désirée.
4. Cliquez sur l'expertise que vous souhaitez ajouter dans la liste filtrée.
5. Pour supprimer une expertise, cliquez sur le bouton "X" situé à droite du nom de l'expertise.
6. Cliquez sur le bouton "ENREGISTRER" pour sauvegarder les modifications.

#### 7 - Gérer l'ORCID
La section Gérer l'ORCID vous fournit un bouton pour enregistrer ou vérifier votre ORCID. Pour en savoir plus sur l'ORCID, veuillez consulter [6.1 - Open Researcher and Contributor Identifier](/fr/guide/advanced-features.html#_6-1-identifiant-du-chercheur-et-du-contributeur-ouvert-orcid).

### 3.3.4 - Sécurité du Compte
![Paramètres Sécurité du Compte](/images/settings_account_security_help.png)

#### 1 - Mot de Passe Actuel
La case Mot de Passe Actuel est une zone de texte pour saisir votre mot de passe actuel si vous essayez de le changer.

#### 2 - Nouveau Mot de Passe
La case Nouveau Mot de Passe est une zone de texte pour saisir votre nouveau mot de passe désiré. Ce mot de passe doit être saisi dans la case Confirmer le Nouveau Mot de Passe également pour garantir l'orthographe correcte. Pour plus d'informations sur la modification de votre mot de passe, veuillez consulter [Section 5.2.2 - Mots de Passe](/fr/guide/customization-and-security.html#_5-2-2-mots-de-passe).

#### 3 - Bouton Afficher le Mot de Passe
Le Bouton Afficher le Mot de Passe affichera brièvement le texte qui se trouve dans la zone de texte du mot de passe associée au bouton d'affichage. Les Boutons Afficher le Mot de Passe sont activés en les cliquant.

#### 4 - Bouton Modifier le Mot de Passe
Le Bouton Modifier le Mot de Passe est le bouton utilisé pour soumettre une demande de changement de mot de passe. Ce bouton est activé en le cliquant.

#### 5 - Historique de l'Authentification
La table Historique de l'Authentification montre toutes les tentatives de connexion à votre compte. Pour plus d'informations sur l'Historique de l'Authentification, veuillez consulter [Section 5.2.3 - Historique de l'Authentification](/fr/guide/customization-and-security.html#_5-2-3-historique-d-authentification).

##### Adresse IP
La colonne Adresse IP est l'adresse de localisation sur Internet associée à l'ordinateur tentant de se connecter à votre compte.

##### Agent
La colonne Agent est le système d'exploitation de l'ordinateur tentant de se connecter à votre compte.

##### Lieu
La colonne Lieu est l'emplacement global approximatif associé à l'adresse IP.

##### Connexion à
La colonne Connexion à est la date et l'heure à laquelle une tentative de connexion à votre compte a été effectuée. L'heure indiquée sera basée sur l'heure de votre ordinateur.

##### Réussie
La colonne Réussie indique si votre compte a été connecté avec succès ou non. En cas de succès, une coche verte sera affichée. En cas d'échec, une croix rouge sera affichée.

##### Déconnexion à
La colonne Déconnexion à indique la date et l'heure à laquelle votre compte a été déconnecté. L'heure indiquée sera basée sur l'heure de votre ordinateur.

### 3.3.5 - Invitations Envoyées

Le Bouton de Menu Utilisateur s'étendra vers le bas pour afficher des boutons de navigation supplémentaires exclusivement pour modifier les options utilisateur.

<img src="/images/button_user_menu_help.png" width="100" height="100">

### 3.4.10 - Bouton Naviguer
Le Bouton Naviguer vous amènera à la page Web associée au bouton.

<img src="/images/button_navigate_help.png" width="100" height="100">

### 3.4.11 - Bouton ORCID
Le Bouton ORCID vous amènera à un formulaire de connexion ORCID. Ce formulaire vous permettra de vérifier et d'associer votre compte ORCID à votre compte sur le Portail de la Science Ouverte.

<img src="/images/button_orcid_help.png" width="250" height="100">

### 3.4.12 - Bouton Sauvegarder
Le Bouton Sauvegarder enverra les modifications que vous avez apportées au serveur du Portail de la Science Ouverte. Le symbole sur le bouton de sauvegarde est une disquette.

<img src="/images/button_floppy_help.png" width="100" height="100">

### 3.4.13 - Bouton Révéler le Texte
Le Bouton Révéler le Texte dévoilera brièvement le texte de votre mot de passe afin que vous puissiez vérifier l'orthographe.

<img src="/images/button_reveal_help.png" width="100" height="100">
