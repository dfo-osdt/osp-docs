# Publication Explorer

![Publication Explorer](./images/publication_explorer.png)

You can explore publications and their details that have been submitted to the OSP. Please note that if a publication is under embargo, only its details can be viewed.

## Filtering {/* #filtering */}

You can filter publications by type using the **Publication Filter Menu** located on the left side of the page.

:::tip

The filtering table is also available for roles with access to the Regional Manuscripts Explorer!

:::

![Filter](./images/publication_explorer_filter.png)

## Download Publications Information {/* #download-publications-information */}

Information on searched publications in the Publication Explorer can be downloaded as a Microsoft Excel (`.xlsx`) file or as a [Research Information Systems (RIS)](https://en.wikipedia.org/wiki/RIS_(file_format)) (`.ris`) file.
:::info
If you do not apply any filters, information on all publications in the OSP will be downloaded!
:::
