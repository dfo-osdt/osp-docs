# Publication Dashboard

The login page features a summary of Publications as they move through the Open Science Portal. You do not need to be logged in to view this page.

To access this page you can do any of the following:
- Go to [https://osp-pso.ent.dfo-mpo.ca/#/](https://osp-pso.ent.dfo-mpo.ca/#/).
- Click the **OSP Home** icon on the left-side of the top banner.
